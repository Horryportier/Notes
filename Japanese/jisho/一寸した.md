# 一寸した
---
1).
,一寸した > ちょっとした,ちょっとした

- slight
- minor
- trifling
- trivial
- petty
**( Pre-noun adjectival (rentaishi); Usually written using kana alone)**
> see also:  [[一寸 ちょっと]]
            
2).
,一寸した > ちょっとした,ちょっとした

- considerable
- proper
- decent
- respectable
- quite (a)
**( Pre-noun adjectival (rentaishi); Usually written using kana alone)**
> see also:  [[一寸 ちょっと]]
