# 未だ未だ
---
1).
,未だ未だ > まだまだ,まだまだ

- still
- much more
- (not) yet
**( Adverb (fukushi); Usually written using kana alone)**
> see also:  [[まだ]]
            
2).
,未だ未だ > まだまだ,まだまだ

- still a long way to go
- still a long way off
- not there yet
**( Expressions (phrases, clauses, etc.); Usually written using kana alone)**
